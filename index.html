Conhecimentos

Android Studio

O Android Studio é o ambiente de desenvolvimento integrado (IDE, na sigla em inglês) oficial para o desenvolvimento de apps para Android e é baseado no IntelliJ IDEA.

Flutter

Flutter é um Framework OpenSource mobile para desenvolvimento de aplicações Android e IOS de alta performance.

Figma

O Figma é uma ferramenta para criação de desenhos de interfaces páginas da web, especialmente aquelas destinadas aos dispositivos móveis.É uma mistura de vários recursos já conhecidos em outros aplicativos de design.

Scrum

O Scrum é uma estrutura ágil para o desenvolvimento, fornecimento e manutenção de produtos complexos, com ênfase inicial no desenvolvimento de software, pesquisa, vendas, marketing e tecnologias avançadas.

Firebase

O Firebase é uma plataforma do Google que contém várias ferramentas e uma excelente infraestrutura para ajudar desenvolvedores web e mobile a criar aplicações de alta qualidade e performance.

GitHub

Um sistema de gerenciamento de projetos e versões de códigos assim como uma plataforma de rede social criado para desenvolvedores
